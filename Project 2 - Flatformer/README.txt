Project: Platformer
Developer: Khoi Ho
Description: Platformer is a simple platform game made with Unity. In this game, the player tries to collect as many coins as possible before the timer reaches 0.
Technology Used: Unity 2019.4.17f1 Personal
Features:
+ Smooth character and camera movement
+ Multiple levels
+ Different types of obstacles: fans, pushing walls, temporary floors, walls that can be disabled by pressing a button
+ Sound effects for jumping, picking up a coin, or pressing a button
+ Simple GUI

How to play:
+ W: move forward
+ S: move backward
+ A: move to the left
+ D: move to the right
